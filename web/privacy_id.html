Kebijakan Privasi

1. Pendahuluan

Selamat datang di MyAnimeDex. Kami menghargai privasi Anda dan berkomitmen untuk melindungi data pribadi Anda. Kebijakan privasi ini akan memberi tahu Anda tentang cara kami mengelola data pribadi Anda saat Anda mengunjungi situs web kami atau menggunakan aplikasi seluler kami dan memberi tahu Anda tentang hak privasi Anda dan bagaimana hukum melindungi Anda.

2. Data yang Kami Kumpulkan

Kami dapat mengumpulkan, menggunakan, menyimpan, dan mentransfer berbagai jenis data pribadi tentang Anda yang telah kami kelompokkan sebagai berikut:

Data Identitas: termasuk nama pengguna dan kata sandi.

Data Penggunaan: mencakup informasi tentang cara Anda menggunakan aplikasi kami, seperti daftar anime, riwayat tontonan, dan favorit.

3. Cara Kami Menggunakan Data Anda

Kami hanya akan menggunakan data pribadi Anda jika hukum mengizinkannya. Paling umum, kami akan menggunakan data pribadi Anda dalam keadaan berikut:

- Untuk menyediakan layanan pelacakan riwayat anime Anda.

- Untuk menyinkronkan data Anda di seluruh perangkat.

- Untuk meningkatkan aplikasi dan pengalaman pengguna kami.

4. Keamanan Data

Kami telah menerapkan langkah-langkah keamanan yang tepat untuk mencegah data pribadi Anda hilang secara tidak sengaja, digunakan atau diakses dengan cara yang tidak sah, diubah, atau diungkapkan.

5. Penghapusan Akun

Anda berhak meminta penghapusan akun Anda dan data terkait. Anda dapat menghapus akun Anda langsung di dalam aplikasi melalui Profil > Zona Bahaya > Hapus Akun. Tindakan ini permanen dan akan menghapus semua data Anda dari server kami.

6. Hubungi Kami

Jika Anda memiliki pertanyaan tentang kebijakan privasi ini atau praktik privasi kami, silakan hubungi kami di support@myanimedex.com.
